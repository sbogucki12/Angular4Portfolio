Entity Framework

Within the .NET environment, I can apply the Entity Framework to build SQL Server databases from C# data models.

From existing databases, I can build entities in order to create APIs to communicate data to a client.

I can use the Entity Framework to create RESTful APIs communicating JSON or XML data.

Using Entity Framework, I can connect Web APIs to SQL Server Databases in order to perform all CRUD / REST (POST/PUT/GET/DELETE) commands.
